# 1. Two Sum

**Difficulty:** Easy
**Topics:** Array

## Complexity

- Time: O(n)
- Space: O(1)

## Follow-ups / gotchas


-Time: O(n): instead of removing index at in place, scan through and find only valid indexs
- brute force: use nums.pop(index). this has a O(n) complexity so it will be like O(n^2) in total
