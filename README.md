# LeetCode Practice

Solutions are organized flat, one folder per problem, numbered by LeetCode problem ID: `NNNN-problem-slug/`.

## Structure

```
leetcode/
  0001-two-sum/
    solution.py
    notes.md
  0002-add-two-numbers/
    solution.py
    notes.md
  template/          # copy this folder to start a new problem
    solution.py
    notes.md
```

## Adding a new problem

1. Copy `template/` to `NNNN-problem-slug/` (zero-padded 4-digit LeetCode ID, kebab-case title).
2. Fill in `solution.py` and `notes.md`.
3. Add a row to the log below.
